## 📞 常見問題

### Q: 為什麼不能全自動部署？
**A:** 需要 GitHub 和 Streamlit 的用戶認證。安全考慮，無法程式方式進行認證。

### Q: 如何快速完成部署？
**A:** 
1. 先在 GitHub 建立空倉庫 (1 分鐘)
2. 執行 `git push -u origin main` (自動)
3. 在 Streamlit 點選部署 (自動)

### Q: 生成的腳本可以重複執行嗎？
**A:** 是的，所有腳本都是冪等的，可以安全重複執行。

### Q: 如何驗證本地準備？
**A:** 執行 `complete_deploy.sh` 進行完整驗證。
